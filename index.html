Projects

Bring it Home

Solo Project | One week

'Bring it Home' enables users to build several fantasy league football teams, within a budget, to compare and contrast the performance of various combinations of players.

I used Django with Python to build out the back-end functionality for users, teams and leagues. Using React on the front-end provides a user-friendly and interactive experience for creating and customising teams.

Out and About

Group Project | One week

'Out and About' encourages people to be more active and social by allowing users to create events and participate in free outdoor activities.

My part in the project included using Express to build the back-end functionality for event attendees and chat boxes, using React to create the homepage and event page and using Mocha and Chai to build 36 back-end tests.

FunFindr

Partner Project | Three days

'FunFindr' is a Node.js app that consumes an Events API to enable users to search for events based on keywords, dates and location.

The site provides search results based on the users preferences and, upon clicking on a specific event, gives detailed information such as date, time and location, including a map.

Pac-Man

Solo Project | 15 days

I built Pac-Man with a focus on replicating the game to the true-life version through ghost-specific AI - each ghost behaves differently in both Chase and Scatter modes - together with ghost Frightened mode where Pac-Man can capture ghosts after eating an Energiser.

PacMan also moves automatically between each key press with CSS animation providing smooth in-game movement.
