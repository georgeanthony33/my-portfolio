Bring it Home

Solo project | 1 week

Bring it Home enables users to create a profile and build several fantasy league football teams within a budget in order to compare between different combinations of players.

React.js on the front-end and Python with Django for the main back-end functionality. I also built a second service using Express.js to make ad-hoc requests for player data from Fantasy Football API and write this to a seed JSON file which in turn populates my PostgreSQL database.

FunFindr

Pair-programming project | 48 hours

FunFindr is a React.js app that consumes a Skiddle Events API for searching events based on keywords, dates and location.

Searching for events takes users to a results page showing available events based on their preferences. From there, users can click on their chosen event to show an event-specific information page for more details including description, date, time, price and location, including a map.

GitHub

Pac-Man

Solo project | 2 weeks

Using solely JavaScript, CSS and HTML, I built a Pac-Man game to be as close as possible to the real version, including the logic for ghost-specific behaviour in both Chase and Scatter modes.

When Pac-Man eats an Energiser, ghosts enter Frightened mode and can be captured and temporarily sent to the pen.

With each level, both the ghost Chase Mode duration and game speed increase. Game speed is handled by CSS animations to ensure movements remain smooth at every level.
